💡 Markdown Token Quick-Reference Guide

Incorporate these exact structural template strings into your message layouts to dynamically fetch values during parsing cycles:

[DATE] - Ticket validation date
[SUSPECT_USERNAME] - Target profile tag
[VICTIM_USERNAME] - Victim context label
[S_REASON] - Suspect violation reason
[V_REASON] - Victim violation context
[MESSAGE] - Operational logs scratchpad
[EVIDENCE] - Dynamic array expanding automatically (handles up to 9 bullet markdown links)
